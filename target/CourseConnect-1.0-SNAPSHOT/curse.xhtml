Contenidos del Curso

Módulo 1: Introducción a las Pruebas Automatizadas

¿Qué son las pruebas automatizadas? Beneficios y desafíos. Herramientas más populares.

Módulo 2: Configuración del Entorno

Instalación de herramientas necesarias (Selenium, JUnit, etc.). Configuración de proyectos.

Módulo 3: Tipos de Pruebas Automatizadas

  • Pruebas unitarias
  • Pruebas de integración
  • Pruebas funcionales

Módulo 4: Creación de Pruebas Automatizadas

Escritura de pruebas efectivas. Uso de aserciones. Mantenimiento de pruebas.

Módulo 5: Mejores Prácticas

Estrategias para mantener la calidad del código. Evitar falsos positivos y negativos. Organización y estructura de pruebas.

Módulo 6: Integración Continua

Conceptos de CI/CD. Herramientas para integración continua.

Módulo 7: Casos de Estudio

Análisis de ejemplos reales. Lecciones aprendidas.

Módulo 8: Proyecto Final

Desarrollo de un proyecto que implemente pruebas automatizadas. Presentación y evaluación del proyecto.
